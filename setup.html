表情アノテーションインターフェース

アノテーションインターフェースを起動

🚀 クイックスタート

  1. 上のボタンをクリックしてアノテーションインターフェースを起動
  2. サンプルCSVファイル（sample_video_list.csv）を読み込み
  3. 動画を再生して表情を確認
  4. 適切な感情を選択
  5. 結果をダウンロード

📁 ローカルファイルを使用する場合

ローカルの動画ファイルを使用する場合は、ローカルサーバーを起動してください：

Python 3の場合：

python -m http.server 8000

Node.jsの場合：

npx http-server

その後、ブラウザで http://localhost:8000 にアクセスしてください。

📋 CSVファイル形式

動画リストのCSVファイルは以下の形式で作成してください：

ID,FilePath,Description video_001,videos/sample1.mp4,Sample emotion video 1 video_002,videos/sample2.mp4,Sample emotion video 2

⌨️ キーボードショートカット

  • 左矢印: 前の動画
  • 右矢印: 次の動画
  • スペース: Pass
  • 1-7: 感情選択（1:怒り、2:嫌悪、3:恐怖、4:幸福、5:悲しみ、6:驚き、7:中立）
